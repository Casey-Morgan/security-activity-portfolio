## Security solution
The security solution I implemented is a **password strength checker** written in Python. The program checks whether a password meets basic security requirements, such as minimum length and the use of different character types.

## What the program does
The program analyses a password and checks for the following:
- minimum length
- uppercase letters
- lowercase letters
- numbers
- special characters

Based on these checks, it gives feedback on whether the password is weak, moderate, or strong. It also tells the user what is missing, so the password can be improved.

## Code

```python
import re

def check_password_strength(password):
    score = 0
    feedback = []

    common_passwords = {
        "password", "123456", "12345678", "qwerty", "abc123",
        "password123", "admin", "letmein", "welcome", "iloveyou"
    }

    # Length checks
    if len(password) < 8:
        feedback.append("Password is too short. Use at least 8 characters.")
    elif len(password) >= 12:
        score += 2
    else:
        score += 1

    # Character variety checks
    if re.search(r"[A-Z]", password):
        score += 1
    else:
        feedback.append("Add at least one uppercase letter.")

    if re.search(r"[a-z]", password):
        score += 1
    else:
        feedback.append("Add at least one lowercase letter.")

    if re.search(r"[0-9]", password):
        score += 1
    else:
        feedback.append("Add at least one number.")

    if re.search(r"[^A-Za-z0-9]", password):
        score += 1
    else:
        feedback.append("Add at least one special character.")

    # Check against common passwords
    if password.lower() in common_passwords:
        feedback.append("This password is too common and easy to guess.")
        score = max(score - 3, 0)

    # Repeated character check
    if re.search(r"(.)\1{2,}", password):
        feedback.append("Avoid repeating the same character too many times.")
        score = max(score - 1, 0)

    # Sequential pattern check
    sequences = ["1234", "abcd", "qwerty", "4321"]
    for seq in sequences:
        if seq in password.lower():
            feedback.append("Avoid common sequences like '1234' or 'abcd'.")
            score = max(score - 1, 0)
            break

    # Leading/trailing spaces
    if password != password.strip():
        feedback.append("Avoid spaces at the beginning or end of the password.")
        score = max(score - 1, 0)

    # Final rating
    if score <= 2:
        strength = "Weak"
    elif score <= 5:
        strength = "Moderate"
    else:
        strength = "Strong"

    return strength, feedback


def main():
    password = input("Enter a password to check: ")
    strength, feedback = check_password_strength(password)

    print(f"\nPassword strength: {strength}")

    if feedback:
        print("Suggestions:")
        for item in feedback:
            print("-", item)
    else:
        print("This password is strong and meets all checks.")


if __name__ == "__main__":
    main()
```

## What the program does
The program checks password strength using several rules, including:
- minimum length
- uppercase letters
- lowercase letters
- numbers
- special characters
- common weak passwords
- repeated characters
- simple sequences such as 1234 or abcd

It then rates the password as weak, moderate, or strong and gives suggestions for improvement.

## Example
Example password checks:
- `password123` → weak
- `Hello123` → moderate
- `H3llo!Secure2026` → strong


## Why this is a security solution
Weak passwords are a common security problem because they are easier to guess or crack. A password strength checker helps users create stronger passwords and reduces the risk of account compromise. This makes it a simple but practical security solution.
