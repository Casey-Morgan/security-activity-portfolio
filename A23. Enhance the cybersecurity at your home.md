# A23. Enhance the cybersecurity at your home

## Description
For this activity, I improved cybersecurity measures at home by reviewing existing security settings and making practical changes to reduce risk. Home cybersecurity is important because personal devices, accounts, and networks are often connected and can all be affected if one part is weak.

## What I did

### 1. Set limited devices to access Wi-Fi and enabled network controls
![network control](https://github.com/user-attachments/assets/538c0b14-37ef-4629-a3f3-5fa984ad7a11)
![Devices](https://github.com/user-attachments/assets/deedbe70-01a0-4900-9868-53827627a45d)

I reviewed which devices were allowed to connect to the home Wi-Fi network and limited access to trusted devices only. This helps reduce the risk of unknown or unnecessary devices joining the network. I also enabled extra network controls, including content filtering and ad blocking. These settings help reduce exposure to unsafe websites, malicious content, and unwanted online tracking. Together, limiting device access and using network controls improve home cybersecurity by reducing possible entry points and making internet use safer.

### 2. Updated device software
![update](https://github.com/user-attachments/assets/a56ec36d-1fd1-4c6c-a61d-641d580d9435)

I checked for software updates on personal devices and made sure the operating system and important apps were updated. Keeping software up to date helps fix known vulnerabilities and improves overall security.

### 3. Enabled Apple Private Relay
![Private relay](https://github.com/user-attachments/assets/f02d2249-b41e-49f9-9f1a-81f24b9318e5)

I enabled Apple Private Relay as an extra privacy and security measure for online browsing. This helps hide my IP address and browsing activity from some network observers and websites. It improves privacy at home by reducing direct exposure of browsing information.

### 4. Checked privacy and security settings on devices
![Tracking](https://github.com/user-attachments/assets/29bf5e88-a340-4e7a-bb4f-faabeea27993)


I reviewed device settings related to screen lock, hidden apps, notification privacy, and other protections. These settings help reduce the chance of someone accessing personal data on the device.

### 5. Created duplicate backups using FreeFileSync
![freefilesync](https://github.com/user-attachments/assets/3651f1c2-d2fa-478b-82da-e75708c87ee6)

I used FreeFileSync to create duplicate backups of important files. This improves home cybersecurity by reducing the risk of data loss if files are deleted, damaged, or affected by a security incident such as ransomware. Having duplicate backups also supports recovery and makes personal data more resilient.


## Why these changes improve security
These changes improve home cybersecurity because they address common risks such as unauthorised network access, outdated software, privacy exposure, device misuse, and data loss. Home security is stronger when network security, device settings, privacy protections, and backup practices are considered together.

## What I learned
This activity showed that improving cybersecurity at home does not always require expensive tools. Small actions such as limiting Wi-Fi access, updating software, using privacy features, and making backups can make a meaningful difference.

## Reflection
By completing this activity, I became more aware of the security risks that exist in a normal home environment. It also helped me understand that cybersecurity at home depends on regular maintenance, not just one-time setup.
