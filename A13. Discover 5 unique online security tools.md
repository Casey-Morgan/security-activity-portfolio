# A13. Discover 5 unique online security tools

## Description
For this activity, I identified five online security tools. Online security tools are software or internet-based services that help protect accounts, devices, networks, and data from cyber threats.


## Selected tools

### 1. VirusTotal
![Virustotal](https://github.com/user-attachments/assets/7d6e867d-6eef-429a-866c-49dbbf21063f)
VirusTotal is an online tool that allows users to scan files, URLs, IP addresses, and domains using multiple security engines. It is useful for checking whether a file or website may be malicious.

### 2. Have I Been Pwned
![Have I been Pwned](https://github.com/user-attachments/assets/f09fe7e0-b929-4561-a42d-c35a015c3bd5)
Have I Been Pwned is an online service that lets users check whether their email address or phone number has appeared in a known data breach. It helps users understand if their accounts may be at risk.

### 3. Google Safe Browsing Transparency Report
![Google](https://github.com/user-attachments/assets/7bf9c049-0ce3-46de-913c-9bc33e8ec7d0)
Google Safe Browsing provides information about whether a website is considered unsafe. It helps users identify sites that may contain malware, phishing content, or other harmful material.

### 4. MXToolbox
![MXT](https://github.com/user-attachments/assets/4c50008d-1b3e-4bd9-8768-70fb404cb303)
MXToolbox is an online tool used to check email and DNS security settings. It can be used to inspect records such as SPF, DKIM, and DMARC, which help protect email systems from spoofing and abuse.

### 5. Qualys SSL Labs
![Qualys](https://github.com/user-attachments/assets/c604149e-3f20-4ed2-9b23-ff226949df20)
Qualys SSL Labs is an online tool that analyses the SSL/TLS configuration of websites. It helps identify weaknesses in HTTPS security and checks whether a site is using modern and secure settings.


## Explanation
These online security tools help users and organisations check for threats, review security settings, and detect risks on the internet. They are useful because they make security testing and awareness more accessible without requiring advanced technical setup.

## Reflection
This activity showed that many useful security tools are available online. They can help with threat detection, account safety, email security, and website security, and they are often easy to use even for beginners.
