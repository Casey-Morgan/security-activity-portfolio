# A7. Discover cryptography used in modern networks

## Description
For this activity, I identified examples of cryptography used in modern networks. In network communication, cryptography is used to protect data while it is being transmitted between devices, servers, and services. It helps prevent unauthorised access, interception, and tampering.

## Selected examples

### 1. TLS in HTTPS
One of the most common examples is TLS used in HTTPS. When a user visits a secure website, TLS helps encrypt the communication between the browser and the web server. This protects sensitive information such as login details, personal data, and payment information.


### 2. WPA2 or WPA3 in Wi-Fi networks
![Wi-Fi ](https://github.com/user-attachments/assets/4b006777-6758-477b-9b4a-1f897aa9b372)

Modern Wi-Fi networks use security protocols such as WPA2 or WPA3. These protocols use cryptographic methods to protect wireless communication and stop unauthorised users from easily accessing the network or reading transmitted data.

### 3. VPN encryption
![VPN](https://github.com/user-attachments/assets/56d88bea-e5ad-43a7-976d-51b9e5b73f8c)

Virtual Private Networks use encryption to protect data sent between a device and the VPN server. This is commonly used on public Wi-Fi or in workplaces to protect traffic from interception.

### 4. SSH for remote access
![SSH](https://github.com/user-attachments/assets/336e61f6-3db2-44ec-bdfb-10f8fc7b8120)
SSH is used for secure remote login and command execution. It encrypts the session between the client and the remote machine. This is much safer than older protocols that sent data in plain text.


### 5. DNS over HTTPS or DNS over TLS
Some modern networks use DNS over HTTPS or DNS over TLS to protect DNS queries. This prevents attackers or third parties from easily viewing or modifying domain name requests.


## Explanation
Cryptography in modern networks protects data in transit. It helps keep communication private, verifies identity, and reduces the risk of interception or manipulation. Without these protections, attackers on the same network or along the communication path could capture sensitive information.

## Reflection
This activity showed that cryptography is a core part of modern networking. It is used in web browsing, wireless access, remote administration, and even background services such as DNS. Most users rely on these protections every day without noticing them.
