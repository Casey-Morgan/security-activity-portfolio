# A11. Discover 5 unique access control devices

## Description
For this activity, I identified five different access control devices used to restrict entry to places, systems, or resources. Access control devices are designed to make sure only authorised people can gain access.

## Selected devices

### 1. Face recognition access terminal
![Face recognition](https://github.com/user-attachments/assets/2baf3c66-f401-4d2e-bee7-785413f5fca4)

A face recognition access terminal uses biometric facial features to verify identity before granting access. It is used in some offices, apartment buildings, and secure facilities. This device controls entry without requiring a physical key or card.

### 2. Boom gate access system
![Boom](https://github.com/user-attachments/assets/f05e734a-c029-426c-8539-16c8736dd874)

A boom gate access system is used in car parks, gated communities, and restricted vehicle entry points. Access is usually controlled by a swipe card, remote control, licence plate recognition system, or security staff approval.


### 3. Electronic hotel keycard lock
![Hotel](https://github.com/user-attachments/assets/3f305d07-674b-478d-8c6d-9c32c374fae8)


An electronic hotel keycard lock allows a guest to enter a room using a programmed card. The card only works for the assigned room and usually only for the booked period. This makes it a practical access control device for temporary authorised access.


### 4. RFID gate or turnstile
![RFID gate](https://github.com/user-attachments/assets/c2719068-11d9-4585-aaee-420e8cb21749)

RFID gates or turnstiles are often used in train stations, office lobbies, and event venues. A user taps a card, ticket, or token, and the gate opens only if the credential is valid.


### 5. Height restriction barrier
![height_restriction_barrier](https://github.com/user-attachments/assets/30e5c071-ac05-4a20-ae63-b09855a5fab0)


A height restriction barrier is a physical access control device placed at the entrance to a car park, driveway, or restricted route. It prevents vehicles above a certain height from entering. This helps protect low-clearance areas and ensures that only suitable vehicles can access the space.

## Explanation
These devices are used to control who can enter a location or use a system. They rely on different methods such as biometrics, cards, building controls, and vehicle entry systems. Although the devices are different, they all support the same goal of preventing unauthorised access.

## Reflection
This activity showed that access control devices are used in many everyday places. It also showed that access can be managed in different ways depending on the level of security needed.
