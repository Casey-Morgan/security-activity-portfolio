# Security Activity Portfolio

Portfolio repository for documenting cybersecurity activities, evidence, reflections, and completed tasks.

## Overview

This repository contains my portfolio work for selected cybersecurity activities.  
It is used to organise evidence, short write-ups, reflections, and any supporting files related to the tasks I completed.

## Purpose

The purpose of this repository is to:

- record completed activities clearly
- store supporting evidence in one place
- show progress across different cybersecurity topics
- present the portfolio in an organised and readable way

## Contents

This repository may include:

- activity summaries
- screenshots and photos
- research notes
- links to tools, articles, or sources
- reflections on what I learned
- code or configuration files for practical tasks

## Activity List

I selected 25 activities from the available portfolio list.  
Each completed activity will have its own section, folder, or file with evidence and a short explanation.

Example structure:

- A1 – Discover security concepts used on campus
- A2 – Discover security concepts used in public space
- A3 – Discover security concepts used in your house
- A5 – Discover cryptographic implementation used online
- A9 – Discover privacy technique used online


## Notes

This repository is maintained for educational and portfolio purposes. Any practical security testing is limited to authorised, controlled, and ethical contexts.
