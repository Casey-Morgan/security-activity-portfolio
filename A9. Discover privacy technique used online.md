# A9. Discover privacy technique used online

## Description
For this activity, I identified privacy techniques used online to protect personal information, browsing activity, and user identity. These techniques help reduce unnecessary data exposure and give users more control over what is collected and shared.

## Selected examples

### 1. Browser private browsing mode
Private browsing mode helps reduce the amount of local browsing data stored on a device, such as history, cookies, and form data after the session ends. It improves privacy on a shared device, although it does not make the user fully anonymous online.

### 2. HTTPS encryption
HTTPS protects data sent between the browser and the website. This helps keep personal information, login details, and browsing activity private from attackers on the network.

### 3. End-to-end encrypted messaging
Messaging services such as WhatsApp or Signal use end-to-end encryption to protect message content. This means only the sender and the receiver should be able to read the messages.

### 4. Two-factor authentication
Two-factor authentication improves account privacy and security by making it harder for someone else to access personal accounts, even if they know the password.

### 5. Cookie consent and tracking controls
Many websites now provide cookie settings and tracking controls. These options let users reject some non-essential cookies and reduce tracking for advertising or analytics purposes.


## Explanation
Privacy techniques used online are designed to limit data exposure, protect communications, and give users more control over their personal information. Some techniques focus on secure communication, while others focus on reducing tracking or preventing unauthorised account access.

## Reflection
This activity showed that online privacy is supported by many small features working together. Some are built into websites and apps, while others depend on user choices, such as turning on two-factor authentication or adjusting tracking settings.
