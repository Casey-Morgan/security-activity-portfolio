# A29. Find a publicly available AI-generated image, video, or audio clip, use at least one detection or verification tool to analyse it

## Description
For this activity, I selected a publicly available image and analysed it using two different methods: the Hive AI detection tool and ChatGPT visual assessment. The purpose was to see whether the image showed signs of being AI-generated and to compare the results from both types of analysis.

## Media selected
Type: Image

## Why I selected it
I selected this image because it had a highly stylised and surreal appearance, which made it suitable for testing with AI detection and verification methods.

## Tools used
- Hive AI detection tool
- ChatGPT visual assessment

## What I did
First, I saved the image and uploaded it to Hive for AI-generated content detection. Hive gave a result indicating whether the image was likely AI-generated.

Second, I used ChatGPT to analyse the visual features of the image. ChatGPT examined the style, lighting, repeated patterns, and overall composition, then gave an assessment of whether the image appeared to be AI-generated.

![Hive](https://github.com/user-attachments/assets/c45ae741-384a-41fe-8fb1-aaca83595411)
![Chatgpt](https://github.com/user-attachments/assets/dcd77fe5-013c-4e05-8c36-628e13e1b9ae)

## Result
Both Hive and ChatGPT suggested that the image was likely AI-generated.

Hive provided a detector-based result, while ChatGPT gave a visual explanation based on the appearance of the image. The visual assessment noted that the image had a surreal fantasy style, repeated rock formations, glowing effects, highly polished lighting, and unusually consistent detail across the scene. These features are commonly associated with AI-generated artwork.


## Analysis
The two methods reached a similar conclusion, which increased confidence that the image was AI-generated. Hive acted as a dedicated detection tool, while ChatGPT acted as a visual assessment tool. Hive provided a more direct classification result, while ChatGPT explained the likely signs of AI generation in plain language.

This comparison showed that different methods can support each other. A detector can provide a structured result, while a visual assessment can explain why the image looks artificial. However, neither method should be treated as perfect proof on its own.

## Limitations
AI detection tools and visual assessments are useful, but they are not always fully accurate. Their results should be treated as indicators rather than absolute proof. A stronger conclusion can be made when multiple methods give similar results.

## Reflection
This activity helped me understand how AI-generated media can be analysed using both automated tools and human-style visual judgement. It also showed that checking suspicious media is more reliable when more than one method is used.
