# A14. Discover 5 AI-enabled security solutions

## Description
For this activity, I identified five security solutions that use artificial intelligence to improve threat detection, investigation, and response. AI-enabled security solutions help security teams analyse large amounts of data, detect suspicious behaviour, and respond to threats faster.

## Selected solutions

### 1. Microsoft Defender XDR
Microsoft Defender XDR is an extended detection and response platform. Microsoft describes it as AI-driven and designed to provide visibility across endpoints, identities, email, cloud apps, and other areas of an organisation.

Source: https://www.microsoft.com/en-au/security/business/siem-and-xdr/microsoft-defender-xdr

### 2. Microsoft Sentinel
Microsoft Sentinel is a cloud-native SIEM platform. Microsoft states that it is powered by AI and automation to support threat detection, investigation, and response across enterprise environments.

Source: https://www.microsoft.com/en-au/security/business/siem-and-xdr/microsoft-sentinel

### 3. Darktrace / EMAIL
Darktrace / EMAIL is an email security solution that uses self-learning AI to detect phishing, impersonation, and other email threats. It is designed to identify unusual behaviour and stop threats that may be missed by traditional filters.

Source: https://www.darktrace.com/products/email

### 4. Palo Alto Networks Cortex XDR
Cortex XDR is a detection and response platform that connects security data from endpoints, cloud, network, identity, and email sources. Palo Alto Networks states that it applies AI to detect and prioritise cyberattacks.

Source: https://www.paloaltonetworks.com/cortex/cortex-xdr

### 5. CrowdStrike Falcon Platform
CrowdStrike Falcon is a cybersecurity platform used for endpoint, identity, cloud, and data protection. CrowdStrike describes it as an AI-native platform that helps detect threats and accelerate investigations.

Source: https://www.crowdstrike.com/en-us/platform/


## Short comparison notes

- **Microsoft Defender XDR** uses AI to automatically disrupt attacks, prioritise incidents, and support analysts with Copilot during investigation and response.  
- **Microsoft Sentinel** uses AI in SIEM operations, including AI-driven SOC optimisation, incident investigation support, query generation, and threat detection enhanced by machine learning.  
- **Darktrace / EMAIL** uses self-learning AI and behavioural analysis to detect unusual email activity, including novel threats that do not rely only on known threat signatures. 
- **Palo Alto Networks Cortex XDR** uses AI to combine data from endpoint, network, cloud, identity, and email sources so it can detect, prioritise, and investigate attacks across multiple vectors. 
- **CrowdStrike Falcon Platform** uses an AI-native architecture built on large volumes of security signals to accelerate investigations, provide real-time intelligence, and automate protection across different domains.

## Reflection
This activity showed that AI is now widely used in commercial cybersecurity platforms. It is especially useful for large-scale monitoring, alert prioritisation, investigation support, and faster response to threats.
